## Figlet

Docker container for [figlet](http://www.figlet.org/) based on [Alpine Linux](https://alpinelinux.org/).

### Build

```sh
docker build --no-cache -t figlet .
```

### Usage 

Usage: 

```sh
docker run --rm figlet Hello world!
```
